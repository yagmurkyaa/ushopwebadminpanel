## 🖥️ Admin Panel

Uygulamanın yönetimini kolaylaştırmak adına geliştirilen web tabanlı admin paneli sayesinde, sistem yöneticileri şu işlemleri gerçekleştirebilir:

- Ürün ekleme, silme ve güncelleme
- Siparişleri takip etme
- Kullanıcı verilerini görüntüleme
- Kategori yönetimi
